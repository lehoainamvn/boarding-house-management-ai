## Tech Stack
- Frontend: HTML, CSS, JavaScript, TailwindCSS
- Backend: Node.js, Express
- Database: SQL Server
- Authentication: JWT
- Machine Learning: Python (scikit-learn)
## Features
- Manage rooms, tenants, and contracts
- Automatic electricity & water billing
- Cost prediction using Machine Learning
- JWT Authentication & Role-based authorization
- Search, filter, pagination
- Internal messaging system

## Architecture
- MVC pattern
- RESTful API design
- Separate frontend & backend

## Getting Started
### Backend
cd bsm-management-backend
npm install
npm start

### Frontend
cd bsm-frontend
npm install
npm start

## AI Cost Prediction
- Algorithm: Linear Regression
- Tool: scikit-learn
- Predict electricity & water cost
